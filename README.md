##  프로젝트 소개

 - 포핀은 위치 기반의 포토카드 거래 서비스입니다.
 - 콘서트, 생일 카페, 팬사인회 등 오프라인 팬 활동 중 자신이 원하는 위치에서 실시간 채팅 서비스를 통해 상대방과 포토카드 거래 일정을 잡고 교환을 할 수 있습니다.

    $ docker-compose build
    $ docker-compose up
